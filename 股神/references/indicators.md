# A股技术分析指标参考

## 常用技术指标速查

### 1. 移动平均线 (MA)

**原理**: N日收盘价的平均值

**常用周期**:
- MA5: 5日线，短线趋势
- MA10: 10日线，短期趋势
- MA20: 20日线，月线，中期趋势
- MA60: 60日线，季度线，长期趋势

**用法**:
- 多头排列: MA5 > MA10 > MA20 > MA60，买入信号
- 空头排列: MA5 < MA10 < MA20 < MA60，卖出信号
- 金叉: 短期MA上穿长期MA，买入信号
- 死叉: 短期MA下穿长期MA，卖出信号

### 2. MACD (指数平滑异同移动平均线)

**原理**: 
- DIF = EMA(12) - EMA(26)
- DEA = EMA(DIF, 9)
- MACD柱 = (DIF - DEA) × 2

**用法**:
- MACD > 0: 多头市场
- MACD < 0: 空头市场
- DIF上穿DEA (金叉): 买入信号
- DIF下穿DEA (死叉): 卖出信号
- 顶背离: 价格新高，MACD未新高，卖出信号
- 底背离: 价格新低，MACD未新低，买入信号

### 3. RSI (相对强弱指标)

**原理**: 比较一段时间内涨跌幅度，衡量买卖力量

**计算公式**: RSI = 100 - (100 / (1 + RS))
- RS = 平均涨幅 / 平均跌幅

**用法**:
- RSI > 70: 超买区，可能回调
- RSI < 30: 超卖区，可能反弹
- RSI在50附近: 震荡整理
- RSI与价格背离: 反转信号

### 4. KDJ (随机指标)

**原理**: 比较收盘价在N日价格区间中的位置

**计算公式**:
- RSV = (收盘价 - N日最低) / (N日最高 - N日最低) × 100
- K = 2/3 × 前K + 1/3 × RSV
- D = 2/3 × 前D + 1/3 × K
- J = 3K - 2D

**用法**:
- K > 80 且 D > 80: 超买
- K < 20 且 D < 20: 超卖
- K上穿D (金叉): 买入信号
- K下穿D (死叉): 卖出信号
- J > 100: 严重超买
- J < 0: 严重超卖

### 5. 布林带 (BOLL)

**原理**: 基于标准差的价格波动区间

**计算公式**:
- 中轨 = MA20
- 上轨 = MA20 + 2 × 标准差
- 下轨 = MA20 - 2 × 标准差

**用法**:
- 价格触及上轨: 可能回调
- 价格触及下轨: 可能反弹
- 布林带收窄: 即将选择方向
- 布林带扩张: 趋势加强

### 6. 成交量 (VOL)

**常用指标**:
- VOL_MA5: 5日均量
- VOL_MA10: 10日均量

**用法**:
- 放量上涨: 上涨趋势确认
- 缩量上涨: 上涨动能不足
- 放量下跌: 恐慌抛售
- 缩量下跌: 抛压减轻

## 综合分析方法

### 趋势判断

1. **多头市场**:
   - 价格在MA20之上
   - MACD > 0
   - RSI在50-70之间
   - 成交量温和放大

2. **空头市场**:
   - 价格在MA20之下
   - MACD < 0
   - RSI在30-50之间
   - 成交量萎缩

### 买卖点判断

**买入信号组合**:
- MACD金叉 + RSI从超卖区回升 + KDJ金叉
- 价格突破MA20 + 成交量放大
- 价格触及布林下轨 + RSI超卖

**卖出信号组合**:
- MACD死叉 + RSI从超买区回落 + KDJ死叉
- 价格跌破MA20 + 成交量放大
- 价格触及布林上轨 + RSI超买

## 新浪股票API数据字段说明

```
数据格式: 股票名称,今日开盘价,昨日收盘价,当前价,今日最高价,今日最低价,
         买一价,卖一价,成交量,成交金额,
         买一到买五(价格和数量),卖一到卖五(价格和数量),
         日期,时间
```

**关键字段**:
- 当前价: 最新成交价
- 涨跌幅: (当前价 - 昨日收盘) / 昨日收盘 × 100%
- 成交量: 当日累计成交股数
- 成交金额: 当日累计成交金额

## 注意事项

1. **技术分析的局限性**:
   - 历史数据不代表未来走势
   - 需要结合基本面分析
   - 单一指标不可靠，需多指标共振

2. **数据来源**:
   - 新浪财经API提供实时数据
   - 历史数据需要从其他渠道获取
   - 数据延迟约15秒

3. **风险提示**:
   - 技术指标仅供参考
   - 不构成投资建议
   - 投资有风险，入市需谨慎
