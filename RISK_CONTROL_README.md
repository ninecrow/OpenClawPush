# 风险控制模块说明

## 解决的核心问题

### 涨停崩盘问题

**原问题：**
- 模型预测某天最高价会涨 9.8%
- 用户按策略挂单买入
- 结果股票涨停，根本无法买入
- 或者更糟：在涨停板上买入，次日低开直接亏损

**风控解决方案：**
```python
if expected_high_return > 0.095:  # 接近涨停
    return BLOCKED, "预测涨幅过大，可能无法成交"
```

## 风控检查项

### 1. 涨停/跌停检测

| 检查项 | 阈值 | 动作 |
|--------|------|------|
| 预期最高价涨幅 | > 9.5% | 禁止买入 |
| 预期最低价跌幅 | < -9.5% | 禁止交易 |
| 当前已涨停 | > 9.5% | 禁止买入 |

**为什么重要：**
- A股涨停后无法继续买入（T+1）
- 即使能买，涨停板买入风险极高
- 避免"买不到"和"高位接盘"两种情况

---

### 2. 波动率异常检测

| 情况 | 判断标准 | 动作 |
|------|---------|------|
| 波动率过高 | 当前 > 平均 × 2.5 | 禁止交易 |
| 波动率过低 | 当前 < 平均 × 0.3 | 谨慎交易 |

**为什么重要：**
- 波动率突增往往伴随重大消息或资金异动
- 模型在极端波动环境下失效概率高
- 低波动率可能是变盘前兆（"暴风雨前的宁静"）

---

### 3. 模型置信度评估

| 指标 | 计算方式 | 作用 |
|------|---------|------|
| 方向准确率 | 最近10次预测方向正确的比例 | 模型当前状态 |
| 平均误差 | 预测价格 vs 实际价格的误差 | 精度评估 |
| 综合置信度 | 方向准确率 × (1 - 误差) | 最终可信度 |

**阈值：** 默认 55%，低于此值禁止交易

---

### 4. 动态仓位管理

仓位计算公式：
```
仓位 = 基础仓位 × 置信度 × 波动率因子

其中：
- 基础仓位：安全=100%, 谨慎=60%, 警告=30%, 危险=0%
- 置信度：模型预测的可靠程度 (0-1)
- 波动率因子：1 / (1 + 波动率 × 10)
```

**最大仓位限制：** 默认 30%
- 即使信号很强，也不超过 30%
- 防止单次交易亏损过大

---

### 5. 止损止盈建议

| 价位 | 计算方式 | 作用 |
|------|---------|------|
| 止损价 | max(买入价×97%, 预测最低价×98%) | 硬止损保护 |
| 止盈价 | min(买入价×105%, 预测最高价×98%) | 获利了结 |

**风险收益比：**
```
R:R = (止盈价 - 买入价) / (买入价 - 止损价)
```

建议只交易 R:R > 1:2 的机会

## 使用方式

### 独立使用风控模块

```python
from risk_control import RiskController

# 初始化
risk_ctrl = RiskController(
    max_daily_return=0.095,    # 涨幅限制
    max_daily_drop=-0.095,     # 跌幅限制
    volatility_threshold=2.5,  # 波动率阈值
    min_confidence=0.55,       # 最低置信度
    max_position=0.3,          # 最大仓位
    stop_loss_pct=0.03,        # 止损 3%
    take_profit_pct=0.05       # 止盈 5%
)

# 评估交易
result = risk_ctrl.evaluate_trade(
    current_price=25.50,
    predicted_high=26.20,
    predicted_low=25.30,
    last_close=25.50,
    current_volatility=0.015
)

print(f"能否交易: {result.can_trade}")
print(f"风险等级: {result.risk_level.value}")
print(f"建议仓位: {result.suggested_position:.0%}")
print(f"止损价: {result.stop_loss_price}")
print(f"止盈价: {result.take_profit_price}")
```

## 交易决策流程图

```
开始
 │
 ├─→ 预测最高价/最低价 ──→ 计算预期涨跌幅
 │                            │
 │                            ▼
 │                       涨跌停检查 ──→ 接近涨停/跌停? ──→ 是 ──→ BLOCKED
 │                            │                           │
 │                            否                          ▼
 │                            ▼                        记录原因
 │                       波动率检查 ──→ 异常波动? ──→ 是 ──→ BLOCKED
 │                            │                           │
 │                            否                          ▼
 │                            ▼                        记录原因
 │                       置信度检查 ──→ 置信度低? ──→ 是 ──→ CAUTION
 │                            │                           │
 │                            否                          ▼
 │                            ▼                        降低仓位
 │                       计算建议仓位
 │                            │
 │                            ▼
 │                       计算止损/止盈价
 │                            │
 │                            ▼
 └─→ 生成交易信号 (BUY/SELL/HOLD/BLOCKED)
```

## 参数调优建议
